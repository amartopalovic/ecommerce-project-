# 🛒 E-commerce Project

## 📌 About

This is a full stack e-commerce project. It includes a frontend built with React and a backend using Node.js and Express.

---

## 🚀 Features

* View products
* Add and remove items from cart
* Cart summary
* Frontend and backend connection

---

## 🧱 Tech Stack

* React
* Node.js
* Express
* JavaScript
* HTML & CSS

---

## 📁 Project Structure

```
react-project/
  ecommerce-project/   (frontend)
  ecommerce-backend/   (backend)
```

---

## ⚙️ How to Run

### 1. Install dependencies

Frontend:

```
cd ecommerce-project
npm install
```

Backend:

```
cd ecommerce-backend
npm install
```

---

### 2. Start the app

Backend:

```
cd ecommerce-backend
npm start
```

Frontend:

```
cd ecommerce-project
npm run dev
```
